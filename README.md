# Shopping-Site
Shopping Web-Site

## Functionalities:
- Admin can add , delete & update products.
- User Registration.
- User can add products to their cart.
- All the Data get stored to Database.
- User can remove products from cart at the timeof billing too.

## Technology Used:
  - Nodejs, Express - BackEnd
  - Html,Css - FrontEnd
  - MongoDb - Database
